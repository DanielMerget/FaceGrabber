Detailed Description

template<class BufferDataType>
class BufferSynchronizer< BufferDataType >

The BufferSynchronizer synchronizes the data that can be pulled from given buffers: The BufferSynchronizer waits until data is available on all buffers. Afterwards, the data is pulled and provided via the publishSynchronizedData signal. BufferSynchronizer is supposed to run in its threading function updateThreadFunc.
