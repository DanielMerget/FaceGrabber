Detailed Description

The OutputStreamsUpdaterStragedy defines method stubs which should called by the KinectHDFaceGrabber for each new FaceFrame. Inherited classes can use the provided data to run various algorithmns. startFaceCollection should be called before calling updateOutputStreams for tracked Kinect-FaceFrame. Afterwards stopFaceCollection should be called. OutputStreamsUpdaterStragedy hides the implementation to allow switching the implementation at run time.
