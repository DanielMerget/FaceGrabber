Detailed Description

Implementation of the OutputStreamsUpdaterStragedy which extracts point clouds using the buffers provided method calls. The processed point clouds are provided without colored. Listeners can registered on cloudsUpdated and cloudUpdated. CloudsUpdated provides the HDFace and RawFaceDepth point clouds. cloudUpdated[0]: HDFace cloudUpdated[1]: RawFaceDepth cloudUpdated[2]: FullRawpDeth.
